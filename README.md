### 使用
双击运行 `app.exe`
> 1. 暂不支持自定义快捷键
> 2. 因隐藏边框后无法使用windows的矿口管理, 所以请先调整好大小后再使用 `ctrl+alt+;` 隐藏边框及菜单
> 3. 快捷键: 
>    * 上一页: `up` || ↑
>    * 下一页: `down` || ↓
>    * 隐藏/显示边框及菜单: `ctrl+alt+;`
>    * 最小化窗口: `esc`
